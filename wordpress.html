Quick Start

WordPress Information

To install this theme you must have a working version of WordPress already installed. If you need help installing WordPress, follow the instructions in WordPress Codex. Below are all the useful links for WordPress information.

  1. WordPress Codex – General info about WordPress and how to install on your server
  2. First Steps With WordPress – General info about multiple topics about WordPress.
  3. FAQ New To WordPress – The most popular FAQ’s regarding WordPress.

Hosting recommendation:

We recommending hosting with cPanel. List of popular providers:

  • Hostgator.com
  • godaddy.com

Theme download url:
http://lpwdev.com/lpw_wordpress/leadingprops.zip

Theme installation:

  1. Download LPW theme
  2. Log in to the WordPress Administration Panels.
  3. Select the Appearance panel, then Themes.
  4. Select Add New.
  5. Use the Upload link in the top links row to up load a zipped copy of a Theme that you have previously downloaded to your machine.

Theme setup:

  1. Activate theme Show image
  2. Install qtranslate Show image
  3. Setup qtranslate Show image

    Supported locales:

    • "en" - "English"
    • "fr" - "Français"
    • "de" - "Deutsch"
    • "ru" - "Русский"

    More locales on demand support@leadingproperties.com

  4. Select the Options -> General panel Be sure that you follow instructions on all tabs.
    • API - URL and Key
    • Header - Logo/Phone/Email ...
    • Footer - information in footer (like address)
    • Social - Social links
    • Google services - Google API key. For Google places service (autocomplete). GET A KEY
    • Styles - Custom styles
    • Scripts - Custom scripts as example analytics scripts (Google analytics)
  5. Default locations can be assigned to show on index page. Show image

Important:

LPW theme creating default pages (Pages panel). This pages created to assign scripts to list property objects from our API, and holding URL routing

Don't delete this pages. You can recreate or reassign this pages in Advanced options, but we aware you from this. Especially if you site is already in production.

Theme development

Be sure that you are using latest version of theme (check theme update in dashboard).

Our platform is on stage of active development and improvements. And our Wordpress theme updating too, to use all features of platform.

We don't recommending to change core files of our theme, because this changes will be rewrited on update.

Wordpress following this way too, and they have solution - Wordpress Child Themes.

Please follow documentation provided by Wordress for Child Theme development. Wordpress Child Themes.

We already create base child theme for easy start. You can download it here.

Child Theme download url:
http://lpwdev.com/lpw_wordpress/lpw-wordpress-child-theme.zip
